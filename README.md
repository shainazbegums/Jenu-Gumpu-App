# 🍯 Jenu-Gumpu App
### Android App Development using GenAI
### Agriculture / Self-Employment Project

---

# 📌 Problem Statement

Tribal and rural honey hunters often sell raw honey to middlemen at very low prices. They lack knowledge about value addition, honey grading, branding, and retail market prices.

Jenu-Gumpu helps honey hunters manage harvests, track floral sources, calculate profits, monitor prices, and improve collective business growth.

---

# 🎯 Project Objective

The main objective of this project is to empower tribal and rural honey producers using Android technology and AI-based analysis tools.

The application helps users:
- Track honey harvest records
- Monitor collective stock
- Calculate profits
- Analyze honey quality
- Track floral sources
- View analytics and charts
- Improve sustainable honey harvesting

---

# ✨ Features

## 🍯 Harvest Log
- Store harvest date
- Store location
- Store honey quantity
- Store floral source
- Generate Batch ID

## 📦 Collective Stock
- View total honey stock collected by the group

## 💰 Profit Calculator
- Calculate estimated profits after filtering costs

## 📊 Analytics Dashboard
- View total stock
- View total batches
- View estimated profit

## 📈 Analytics Charts
- Monthly honey collection chart visualization

## 📸 AI Honey Scanner
- Simulated honey quality analysis using AI concepts

## 💵 Price Monitor
- Compare wholesale and retail honey prices

## 🌱 Sustainable Harvest Guidelines
- Eco-friendly honey harvesting practices
- Bee colony protection awareness

## 🌐 Kannada Language Support
- Dashboard includes Kannada labels for rural accessibility
